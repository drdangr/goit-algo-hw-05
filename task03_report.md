# Порівняння алгоритмів пошуку підрядка

## Постановка задачі

Метою роботи є експериментальне порівняння трьох алгоритмів пошуку підрядка:
- Боєра–Мура (Boyer–Moore),
- Кнута–Морріса–Пратта (KMP),
- Рабіна–Карпа (Rabin–Karp)

на основі двох текстів (стаття 1 та стаття 2) для двох випадків:
1. підрядок існує в тексті;
2. підрядок відсутній у тексті.

## Методика дослідження

- Для кожного тексту було обрано:
  - один реальний підрядок, який гарантовано міститься в тексті;
  - один вигаданий підрядок, якого немає в тексті.
- Для вимірювання часу виконання використано модуль `timeit`.
- Для кожної комбінації («текст» × «алгоритм» × «тип підрядка») виконувалося по **10 запусків**.
  У звіт заносився **середній час**.

# Вмсновки

1.	**Boyer–Moore**: Найшвидший алгоритм у всіх тестах.
2.	**KMP (Кнута–Морріса–Пратта)** займає «середню» позицію.
3.	**Rabin–Karp** виявився найповільнішим.
4.	Отримані результати узгоджуються з теоретичними оцінками:
    • Boyer–Moore ефективно «перестрибує» частину символів завдяки правилам зсуву,
тому на великих текстах і відносно довгих патернах він дає найкращий час;
	• KMP забезпечує лінійну від довжини тексту складність, але часто повільніший за Boyer–Moore;
	•Rabin–Karp має зручну ідею з хешуванням, але витрати на обчислення й
перевірку хешів роблять його повільнішим (у цьому конкретному порівнянні).

Таким чином, для розв’язання задачі пошуку однієї підрядка в окремому тексті
найкращим вибором у проведеному експерименті став алгоритм **Боєра–Мура**.
